# Multi-City Weather Analytics Dashboard

A real-time Power BI dashboard tracking live weather conditions and air quality across multiple Indian cities (Mumbai, Noida, Lucknow), built with dynamic DAX measures and a live API data feed.

## Features
- Live weather cards (temperature, humidity, pressure, visibility, UV index)
- City-switcher panel for instant comparison across locations
- 7-day forecast trend chart
- Air Quality Index gauge (CO, O3, PM10, PM2.5, NO2, SO2) with color-coded health status
- Rain-probability tracker and sunrise/sunset panel

## Tech Stack
Power BI · DAX · Weather API integration
